# Portafolio web personal

Portafolio moderno y responsive creado con HTML, CSS y JavaScript puro para presentar el perfil de un desarrollador semisenior enfocado en backend, calidad de software, bases de datos, cloud y aprendizaje continuo.

## Estructura

- index.html
- css/styles.css
- css/responsive.css
- js/main.js
- js/animations.js
- assets/img
- assets/icons
- assets/projects

## Características

- Diseño oscuro, tecnológico y elegante
- Secciones: inicio, sobre mí, habilidades, proyectos, experiencia, aprendizaje, contacto
- Menú responsive con hamburguesa
- Animaciones de aparición y typing
- Filtro de proyectos
- Validación básica del formulario
- Botón para volver arriba
- Tema alternativo opcional

## Uso

Abre index.html en tu navegador o sirve la carpeta con un servidor local simple.
